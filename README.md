## Mini E-commerce:

Aplicação web de um mini e-commerce com listagem de produtos e carrinho de compras, desenvolvida com JavaScript puro e foco em uma interface moderna e responsiva.

## Funcionalidades

* Listagem dinâmica de produtos
*  Adicionar produtos ao carrinho
*  Página de carrinho
*  Remover itens do carrinho
*  Cálculo do total da compra
*  Contador de itens no carrinho
*  Feedback visual (toast ao adicionar produto)
*  Persistência com LocalStorage

## Tecnologias utilizadas

* HTML5
* CSS3 (com variáveis CSS)
* JavaScript (ES6+)
* LocalStorage

## Objetivo do projeto

Este projeto foi desenvolvido para praticar:
* Manipulação do DOM
* Renderização dinâmica de elementos
* Gerenciamento de estado (carrinho)
* Organização de código em múltiplos arquivos
* Criação de interfaces modernas

## Aviso

Este projeto utiliza armazenamento local (LocalStorage) e não possui backend.
Não deve ser usado em produção.

## Aprendizados

* Estruturação de aplicações frontend
* Uso de arrays e objetos em JavaScript
* Persistência de dados no navegador
* Separação de responsabilidades (products, cart, main)
* Melhoria de UX com feedback visual
